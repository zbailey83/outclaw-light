# 30-Day Organic Marketing Blitz — Outclaw Sprint Calendar

This is the prioritized execution plan. Do not try to do everything at once. Follow the sequence.

---

## Pre-Sprint Setup (Day 0)

Before anything else:
- [ ] Set up Google Alerts for 5–10 keywords (see `references/outreach.md`)
- [ ] Create Twitter search saved searches for competitor complaints + "AI tools" queries
- [ ] Set up a simple tracking spreadsheet for outreach (name, platform, status, follow-up date)
- [ ] Identify 5 target subreddits where Outclaw's audience lives
- [ ] List 5 competitor newsletters + 3 competitor tools to mine for reviews

---

## Week 1 — Reddit + Competitor Mining (Days 1–7)

**Goal:** First subscribers from warm leads. Test which Reddit formats work.

| Day | Primary Task | Secondary Task |
|-----|-------------|----------------|
| 1 | Write + publish first Reddit value post (3–5 subreddits) | Set up competitor review search on G2/Capterra |

**Week 1 output targets:** 2 Reddit posts live, 15+ DMs sent, 1 comparison page brief written

## Month 2+: The Compound Phase

By Day 30, you should know which 3–4 channels are working. Double down on those. Cut the rest.

The channels that compound over time:
- **SEO blog posts** — keep publishing 1/week
- **Reddit** — keep posting 2x/week
- **Content recycling** — keep the pipeline running
- **Build in public** — never stop

The channels that require ongoing manual effort but convert well:
- Competitor review mining — run once/week
- Community participation — daily engagement

Pick your battles. Do the compounding channels consistently. Do the manual channels at sustainable volume.
