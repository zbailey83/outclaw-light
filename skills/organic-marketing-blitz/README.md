## Outclaw Brand Positioning

Apply this to every piece of output across every channel. Every Reddit post, DM, and tweet must reflect this identity.

**Who we help:** Marketers, founders, operators, and builders actively using AI tools

**What we are:** The sharpest, most practical AI tools newsletter — not a news aggregator, not a beginner blog

**What we're not:** A generic "AI news" account. We have opinions. We do hands-on testing. We call out hype.

**One-line pitch:**
> *"Outclaw covers what actually works in AI tools — tested, opinionated, no fluff."*

**Brand voice:** Sharp. Practical. Opinionated.

> ⚠️ Never pitch "subscribe to our newsletter." Always pitch the specific value they'd get from it.

## Execution Rules (Non-Negotiable)

When executing any channel, the skill must:

1. **Read the relevant reference file first** before generating any output
2. **Apply Outclaw brand voice** — Sharp. Practical. Opinionated.
3. **Generate real, ready-to-use outputs** — actual post copy, actual DM scripts, actual page outlines. Not summaries or suggestions.
4. **Include the failure mode** — what people get wrong about this channel, and how Outclaw avoids it
5. **Include cadence expectations** — what "doing this right" looks like daily/weekly

## Channel Failure Modes

| Channel | What People Get Wrong |
|---|---|
| Reddit | Posting links without providing value first → instant ban / ignored |
| Competitor review mining | Pitching too soon before understanding the specific pain |
| SEO comparison pages | Writing for Google instead of the person ready to switch tools |
| Content recycling | Copy-pasting instead of reformatting for each platform's native style |
| Cold DM | Leading with the product instead of the problem they're experiencing |
| Build in public | Posting metrics nobody cares about instead of decisions and learnings |
| Product Hunt | Launching without warming up the community first |
