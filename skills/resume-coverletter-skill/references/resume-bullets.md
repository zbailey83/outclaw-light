# Resume Bullet Rewrites — Before / After by Role

## Core Rules (apply to all roles)
- Start with a strong past-tense action verb
- Quantify: numbers, %, volume, time saved, scale
- No "Responsible for", "Helped with", "Worked on"
- No nominalizations ("the creation of" → "created")
- No adverbs — find a stronger verb instead
- One idea per bullet. If it needs two verbs to describe one outcome, fine. If it's two separate things, split it.

**Formula**: `[Verb] + [what] + [tool/method] + [outcome or scale]`

## Verb Bank (strong, specific, never-vague)

**Building / Creating**: Built, Shipped, Launched, Wrote, Designed, Developed, Authored, Coded, Produced, Published

**Improving**: Reduced, Cut, Improved, Grew, Increased, Raised, Lifted, Accelerated, Streamlined

**Leading / Managing**: Led, Directed, Ran, Oversaw, Coordinated, Partnered, Collaborated

**Analyzing**: Analyzed, Audited, Mapped, Identified, Tracked, Measured, Benchmarked

**Maintaining**: Maintained, Updated, Revised, Fixed, Patched, Held, Sustained

**Documenting**: Documented, Wrote, Structured, Organized, Categorized, Archived

## What to Never Write

| ❌ Never | ✅ Always |
|---------|---------|
| Responsible for | Built / Led / Wrote / Ran |
| Helped with | [Just say what you did] |
| Assisted in the creation of | Created |
| Worked on | [Name the specific thing] |
| Utilized | Used |
| Facilitated | [Name the actual action] |
| Leveraged | Used / Applied |
| Synergy / Synergized | [Delete entirely] |
| Spearheaded | Led |
| Orchestrated | Coordinated / Ran |
| Impactful | [Show the impact with a number] |
| Empower / Empowered | [Show what actually changed] |
